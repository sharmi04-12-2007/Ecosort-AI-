# EcoSort AI ♻️

AI-Powered Smart Waste Segregation and Recycling Assistant

## Overview

EcoSort AI helps users identify waste types through image classification and provides appropriate disposal recommendations. The project promotes sustainable waste management and supports the United Nations Sustainable Development Goals (SDG 11, SDG 12, and SDG 13).

## Problem Statement

Improper waste segregation leads to:
- Increased landfill overflow
- Reduced recycling efficiency
- Environmental pollution
- Lack of awareness regarding proper disposal methods

## Solution

EcoSort AI uses Artificial Intelligence and Computer Vision to:

1. Accept waste images from users
2. Classify waste into categories
3. Recommend appropriate disposal methods
4. Provide sustainability tips
5. Track environmental impact

## Features

- Image-based waste classification
- Smart disposal recommendations
- Recycling guidance
- Sustainability tips
- Environmental impact tracking
- User-friendly interface
